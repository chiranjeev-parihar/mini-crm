# UI Guidelines

Version: 1.0

\---

# Objective

Maintain a professional and consistent CRM interface.

Every new module must follow these guidelines.

\---

# Design

Simple

Professional

Business Focused

Responsive

\---

# Cards

Rounded corners

Consistent padding

Consistent spacing

\---

# Tables

Search

Filters

Pagination

Responsive

Loading

Empty State

\---

# Forms

Required fields marked.

Validation below fields.

Primary button at bottom.

Cancel button secondary.

\---

# Dashboard

Always display

Live Data

Today's Work

Quick Actions

Statistics

Recent Activity

\---

# Empty States

Never leave blank pages.

Examples

No Leads Found

No Tasks Available

No Customers Yet

\---

# Icons

Use one icon library only.

Maintain consistency.

\---

# Mobile

Every page must support

Desktop

Tablet

Mobile

\---

# UX Rules

Never allow destructive actions without confirmation.

Delete buttons should not appear for:

Leads

Customers

Tasks

Follow-ups

Use Archive or Cancel workflows.

\---

# Navigation

Sidebar consistent.

Breadcrumbs where useful.

Current page highlighted.
