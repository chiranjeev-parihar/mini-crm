# Mini CRM

Version: 1.0

## Purpose

Mini CRM is a lightweight Customer Relationship Management system designed for small businesses to manage leads, customers, follow-ups, quotations, and sales activities.

## Goal

Build a reusable CRM template that can be customized for different industries such as Solar, Real Estate, Insurance, Manufacturing, Education, and Service businesses.

## Target Users

* Sales Executive
* Sales Manager
* Admin

## Core Modules

* Authentication
* Dashboard
* Leads
* Customers
* Follow-ups
* Tasks
* Notes
* Quotations
* Reports
* Settings

## Future Modules

* WhatsApp Integration
* AI Follow-up Assistant
* Voice Calling
* AI Lead Scoring
* Workflow Automation
* Email Integration
* Payment Tracking

## Success Criteria

* Easy to customize
* Responsive
* Fast
* Clean UI
* Modular Architecture
