# Customer Management Module ‚Äî Implementation Plan

## Overview

Implement the **Customer Management** module for Mini CRM. Customers are **only** created by converting a Won Lead. The module provides full CRUD for customer records (except Create directly), lead-to-customer conversion with duplicate protection, and a polished frontend UI matching the existing design language.

---

## Architecture Summary

| Layer | Pattern |
|---|---|
| Database | Prisma + SQLite (existing) |
| Backend | Express + TypeScript, Controller ‚Üí Service ‚Üí Prisma |
| Frontend | React + TypeScript + Vite + TailwindCSS + TanStack Query |

---

### Database ‚Äî Prisma Schema

#### [MODIFY] [schema.prisma](file:///f:/Project_Ishita/Portfolio/MiniCRM/database/prisma/schema.prisma)

Add to the **Lead** model:
- `isConverted  Boolean   @default(false) @map("is_converted")`
- `convertedAt  DateTime? @map("converted_at")`
- `customer     Customer?` (relation)

Add new **Customer** model:
```prisma
model Customer {
  id            String   @id @default(uuid())
  leadId        String   @unique @map("lead_id")
  company       String?
  contactPerson String?  @map("contact_person")
  phone         String?
  email         String?
  gstNumber     String?  @map("gst_number")
  address       String?
  industry      String?
  notes         String?
  createdAt     DateTime @default(now()) @map("created_at")
  updatedAt     DateTime @updatedAt @map("updated_at")
  lead          Lead     @relation(fields: [leadId], references: [id])
  @@map("customers")
}
```

## Verification Plan

### Migration Command
```bash
cd database && npx prisma migrate dev --name add_customer_conversion
```

### Automated Tests (Manual Steps)
1. **Convert Won Lead** ‚Üí Customer record created, Lead marked `isConverted=true`
2. **Convert again** ‚Üí API returns 409 Conflict
3. **Convert non-Won lead** ‚Üí API returns 400 with correct message
4. **GET /customers** ‚Üí Lists converted customers with search/pagination
5. **GET /customers/:id** ‚Üí Returns single customer
6. **PUT /customers/:id** ‚Üí Updates customer
7. **DELETE /customers/:id** ‚Üí Removes customer
8. **Frontend**: Won lead shows "Convert to Customer" button
9. **Frontend**: After conversion, button replaced by "Converted" badge
10. **Frontend**: Customer List, View, Edit pages all functional
